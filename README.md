# Portfolio

Personal developer portfolio built with **React + TypeScript**, showcasing professional experience, projects, and skills.

The application supports **internationalization (i18n)**, **light/dark theming**, and responsive layouts, focusing on clean architecture and maintainable code.

## Tech Stack

- **Node.js**: 20.x  
- **React**: 18.x  
- **TypeScript**: 5.x  
- **Vite**: 5.x  
- **Tailwind CSS**: 3.x  
- **react-i18next**: 13.x
